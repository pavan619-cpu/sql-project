# SQL Project

##  Project Overview
This project demonstrates basic to slightly intermediate SQL queries using an `employee` table.  
It is designed to showcase understanding of data filtering, aggregation, sorting, and subqueries.

This project is suitable for **SQL beginners building their first GitHub portfolio**.

---

##  Database Structure

### Table: `employee`

| Column Name | Data Type | Description |
|------------|----------|-------------|
| emp_id     | INT (PK) | Employee ID |
| name       | VARCHAR  | Employee Name |
| salary     | INT      | Employee Salary |
| branch_id  | INT      | Branch Identifier |

---

##  SQL Concepts Covered

- CREATE TABLE
- INSERT INTO
- SELECT
- WHERE
- BETWEEN
- ORDER BY
- GROUP BY
- HAVING
- Aggregate Functions (COUNT, SUM, AVG)
- Subqueries
- LIKE operator
- LIMIT

---

##  Queries Included

- Employees earning within a salary range
- Sorting employees by salary
- Counting employees per branch
- Branches with average salary above a condition
- Highest paid employee
- Pattern matching using LIKE
- Employees earning more than average salary (subquery)
- Total salary paid per branch

##  How to Run This Project

1. Open **MySQL Workbench**
2. Execute `schema.sql` to create the table
3. Execute `data.sql` to insert sample data
4. Run queries from `queries.sql` to view results
